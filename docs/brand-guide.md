## 品牌定位

面向所有 Chrome 用户的 AI 浏览器助手。不需要安装本地应用、不需要 Node.js/Python 运行时——装个 Chrome 扩展，用自然语言就能操控网页。从简单的填表点按钮，到连接本地和远程机器执行复杂任务，一个入口搞定。

## 名字由来

"白雪"取自丁丁历险记（The Adventures of Tintin）中的小狗 Snowy（法语名 Milou）。它忠诚、机灵，跟着主人到处冒险——跟浏览器 Agent 的定位完美契合：

- **忠诚**：始终在你的浏览器里待命
- **机灵**：能理解页面、操作元素、帮你做事
- **冒险伙伴**：陪你一起浏览、操作、解决问题

## Tone of Voice

### 原则

**双面型** — 入门时亲切简洁，展示能力时专业可靠。

### 用词规范

| 做 | 不做 |
|----|------|
| "告诉白雪你想做什么" | "通过自然语言界面输入指令" |
| "帮你搞定" | "实现端到端的网页自动化" |
| "悄悄在后台帮你干活" | "基于 Service Worker 的后台执行引擎" |
| "装上就能用" | "零配置的轻量级部署方案" |
| "浏览器里的 Linux" | "基于 WebAssembly 的沙盒化 POSIX 环境" |

### 角色感

对话用户时，像一只聪明的小狗坐在你旁边等你下令——亲切但不谄媚，自信但不傲慢。

## 视觉方向（待定）

- **Logo**：白色小狗剪影/图标，线条简洁，现代感，致敬丁丁但不直接用原图
- **配色**：主色冷白 + 一个亮色点缀，呼应 Chrome 扩展图标体系
- **域名建议**：`getsnowy.com`（首选）、`snowy.ai`、`snowy.dev`

## 竞品差异化

白雪的核心差异化不是"不上云"（LLM 调用仍走云端 API），而是**不需要装任何本地应用**：

| 对比项 | 白雪 Snowy | 其他浏览器自动化工具 |
|--------|-----------|-------------------|
| 安装方式 | Chrome 扩展，一键安装 | 需装 Node.js / Python / 桌面应用 |
| 运行环境 | 浏览器 Service Worker | 本地服务器 / 桌面进程 |
| 配置复杂度 | 零配置 | 环境配置、依赖安装、API 配置 |
| 交互方式 | 自然语言对话 | 代码 / 脚本 / 配置文件 |
| 扩展能力 | Skill / Plugin / MCP | 取决于具体工具 |
